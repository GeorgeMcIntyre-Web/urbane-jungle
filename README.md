# Urbane Jungle - Luxury E-commerce

A high-end Next.js 16 e-commerce platform for premium indoor plants.

## Tech Stack
- Next.js 16.0.3 (App Router)
- React 19.2.0
- TypeScript
- Tailwind CSS
- Drizzle ORM + SQLite
- Stripe (payments)
- Framer Motion (animations)

## Development
```bash
pnpm install
pnpm dev
```

## Production Build
```bash
pnpm build
pnpm start
```

## Deployment
The site is configured for Cloudflare Pages deployment.

Build settings:
- Build command: `pnpm build`
- Build output directory: `.next`
- Node version: 18+

## Features
- Server-side rendering
- Optimized images
- Cart management
- Stripe checkout integration
- Responsive design
- Luxury animations
